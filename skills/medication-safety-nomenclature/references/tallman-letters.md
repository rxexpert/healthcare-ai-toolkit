# FDA and ISMP Tall Man Letter Reference

This reference contains the complete FDA-approved and ISMP-recommended lists of look-alike drug names with tall man lettering. Apply these consistently across all medication documentation.

## How to Use This List

- **Bold and UPPERCASE** the designated letters in look-alike drug name pairs
- Apply to ALL instances of these drug names in any document
- When both brand and generic names appear, apply tall man letters to both where applicable
- Brand names are marked with * in the ISMP list

## Notes

- Brand names are indicated with *
- The ISMP list is for voluntary use and is not FDA-approved
- Some drugs appear on both FDA and ISMP lists
- Apply tall man letters with **bold** formatting in addition to capitalization
- When formatting is limited (plain text), use UPPERCASE only

## Quick Reference for Most Common Pairs

**High-Alert Medications:**
- vinBLAStine / vinCRIStine
- DOBUTamine / DOPamine  
- HYDROmorphone / morphine
- EPINEPHrine / ePHEDrine

**Frequent Confusions:**
- predniSONE / prednisoLONE
- glipiZIDE / glyBURIDE
- ALPRAZolam / LORazepam
- metFORMIN / metroNIDAZOLE
- HumaLOG / HumuLIN

**Source:** FDA and ISMP Lists of Look-Alike Drug Names with Recommended Tall Man Letters (2016)
